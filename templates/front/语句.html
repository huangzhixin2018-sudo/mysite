基础语句

赋值语句

用于将值赋给变量，Python支持多种赋值方式。

x = 10 # 简单赋值 a, b = 1, 2 # 多重赋值 x += 5 # 增强赋值 name, age = "Alice", 25 # 同时赋值多个变量

条件语句 (if/elif/else)

根据条件执行不同的代码块。

if x > 10: print("x大于10") elif x == 10: print("x等于10") else: print("x小于10") # 单行条件表达式 result = "大于10" if x > 10 else "不大于10"

循环语句 (for/while)

用于重复执行代码块。

for循环

# 遍历序列 for item in [1, 2, 3]: print(item) # 使用range for i in range(5): # 0到4 print(i) # 遍历字典 for key, value in {"a": 1, "b": 2}.items(): print(key, value)

while循环

count = 0 while count < 5: print(count) count += 1 # 无限循环 while True: print("循环中...") break # 实际应用中应该有退出条件

控制流语句

break/continue

控制循环的执行流程。

break示例

for i in range(10): if i == 5: break # 退出整个循环 print(i)

continue示例

for i in range(5): if i == 2: continue # 跳过当前迭代，继续下一次 print(i)

pass语句

空操作语句，用于占位。

if x < 0: pass # 什么也不做，占位用 else: print("x是正数") # 在类或函数定义中作为占位 class MyClass: pass # 暂时不实现 def my_function(): pass # 待实现

异常处理

try/except/finally

捕获和处理异常。

try: result = 10 / 0 except ZeroDivisionError: print("不能除以零") except (TypeError, ValueError) as e: print(f"类型或值错误: {e}") except Exception as e: print(f"发生未知错误: {e}") else: print("没有发生异常") finally: print("无论如何都会执行的清理操作")

raise语句

主动抛出异常。

if x < 0: raise ValueError("x不能为负数") # 重新抛出当前异常 try: dangerous_operation() except SomeError: print("处理部分错误") raise # 重新抛出 # 链式异常 try: open("nonexistent.txt") except FileNotFoundError as e: raise RuntimeError("文件处理失败") from e

函数相关

def语句

定义函数。

# 基本函数定义 def greet(name): return f"Hello, {name}!" # 带默认参数的函数 def power(base, exponent=2): return base ** exponent # 可变参数 def sum_all(*args): return sum(args) # 关键字参数 def person_info(**kwargs): for key, value in kwargs.items(): print(f"{key}: {value}")

return语句

从函数返回值。

# 返回单个值 def add(a, b): return a + b # 返回多个值(实际上是返回元组) def min_max(numbers): return min(numbers), max(numbers) # 提前返回 def is_positive(num): if num <= 0: return False return True # 无返回值(实际返回None) def print_hello(): print("Hello")

yield语句

定义生成器函数。

# 简单的生成器函数 def count_up_to(n): i = 1 while i <= n: yield i i += 1 # 使用生成器 for num in count_up_to(5): print(num) # yield from语法 def chain(*iterables): for it in iterables: yield from it

导入语句

import语句

导入模块和包。

基本导入

import math import numpy as np import os.path

从模块导入特定内容

from math import sqrt, pi from collections import defaultdict as ddict from pathlib import Path

导入所有内容(不推荐)

from math import * # 导入math模块所有内容

相对导入(在包内使用)

from . import module1 # 从当前包导入 from ..subpackage import module2 # 从上级包的子包导入
