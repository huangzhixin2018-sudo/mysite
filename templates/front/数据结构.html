列表(List)

基本概念

列表是Python中最基本的数据结构之一，它是一个有序的元素集合，可以包含不同类型的元素。

创建列表

使用方括号[]创建列表：

my_list = [1, 2, 3, 'a', 'b', 'c']

常用操作

访问元素：使用索引my_list[0]

修改元素：my_list[0] = 'new value'

添加元素：my_list.append('new item')

删除元素：del my_list[0]或my_list.remove('a')

切片操作：my_list[1:3]获取索引1到2的元素

高级用法

列表推导式：[x*2 for x in range(10)]

排序：my_list.sort()或sorted(my_list)

集合(Set)

基本概念

集合是一个无序的不重复元素集，常用于成员测试和消除重复元素。

创建集合

使用set()函数或花括号{}创建集合：

my_set = {1, 2, 3, 3, 4} # 结果为{1, 2, 3, 4}

常用操作

添加元素：my_set.add(5)

删除元素：my_set.remove(3)

集合运算：并集|，交集&，差集-

成员测试：2 in my_set

高级用法

冻结集合：frozenset创建不可变集合

集合推导式：{x for x in 'abracadabra' if x not in 'abc'}
